演進譜系

清單與作用

名稱類型作用
Centerline入門·走線經典 DeepRacer 起手式。依離中線距離給分（很近 1.0、稍偏 0.5、再偏 0.1、離道極低）。只練貼中線。
Centerline + speed入門·走線+速度貼中線給 1.0，且在道上時依速度額外加分。中線 + 鼓勵快。
Apex 3P走內側線彎中走內側車道（apex line）：用前幾個 waypoint 曲率判左/右彎，左彎走左道、右彎走右道 → 1.0，否則 ~0。
Apex lane (waypoint curve)走內側線與 Apex 3P 幾乎相同的內側車道邏輯。兩者近乎重複。
Apex Off-Road走內側線·壓線Apex 內側車道，且「切內側切到出界」再給 2.0（獎勵壓線切 apex）。
Apex Speed走內側線+速度彎中走內側（出界切 apex 給 2.0）＋ 直線（前 10 點對齊時）獎勵速度 speed²/8。
Apex Brake Speed速度/煞車區首呼叫算「左彎/右彎/入彎口/煞車區」四表：走對車道給基底，煞車區降速、其餘衝速。已知毛病：全程煞車項會壓速度。
Tangent Apex ⭐切點轉向切點行為獎勵：車頭瞄準下一彎切點，只獎勵「把車頭轉回切點的正確轉向修正動作」（二元 0/1）。純轉向、不含速度，需搭配速度獎勵。
Tangent Apex Speed ⭐切點+速度Tangent Apex 的切點轉向 ＋ 區間速度：入彎前煞車 2.0 → 彎中 2.5 → 後三點全速出彎 → 直線衝頂 4.0。reward = steer + v（0–2）。可單獨實戰跑圈。
重複提醒：Apex 3P ≈ Apex lane (waypoint curve)（邏輯近乎相同）。
設計教材：Tangent Apex 規格 · Tangent Apex Speed 規格。

獎勵函數皆為單檔、純標準庫；訓練時其原始碼會凍結複製進各模型的 config，故更動／刪除 library 檔不影響已訓練模型。
