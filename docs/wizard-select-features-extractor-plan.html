UI onlyopt-inenv.yaml + meta詳細頁一列3 任務 TDD

目標：在「建立模型」精靈讓使用者選 原本 NatureCNN 或 迷你 TinyCNN，選值寫進該模型的 env.yaml（訓練時 build_model 自動切網路）；詳細頁多一列顯示用哪個網路。
後端已就緒：Config.features_extractor 是合法 env.yaml key、load_config 自動接受、build_config_dir 有 extra_env、build_model 已會切換 → 只動精靈 UI + 詳細頁一列。

資料流

全域限制

  • 只動 topracer/ui/wizards/create_model_wizard.py 與 topracer/ui/pages/model_details.py（＋各自測試）。不改後端。
  • combo 值固定 "nature" | "tiny"，預設 nature；標籤「原本 NatureCNN（預設）」/「迷你 TinyCNN（~200× 小）」。
  • 詳細頁列名「策略網路」；值「原本 NatureCNN」/「迷你 TinyCNN」；缺鍵（舊模型）→「原本 NatureCNN」。
  • 這些是 Qt GUI 測試：macOS 要 QT_QPA_PLATFORM=cocoa。整輪：QT_QPA_PLATFORM=cocoa .venv/bin/python -m pytest tests/ui/test_create_model_wizard.py tests/ui/test_model_details.py -v。

T2建模精靈：選 NatureCNN / TinyCNN

檔案
  • 修改 topracer/ui/wizards/create_model_wizard.py
  • 修改 tests/ui/test_create_model_wizard.py
介面
  • Consumes：既有 build_config_dir(extra_env=…)、Config.features_extractor、load_config
  • Produces：
    • net_combo + features_extractor()
    • build_spec 把選值寫進 env.yaml（extra_env）與 meta
① 寫失敗測試
def test_features_extractor_flows_into_spec(qtbot, tmp_path):
    w = _wiz(qtbot, tmp_path)
    assert w.features_extractor() == "nature"          # 預設（combo index 0）
    w.net_combo.setCurrentIndex(1)                      # tiny
    w.name_edit.setText("m-tiny")
    spec = w.build_spec()
    assert spec["meta"]["features_extractor"] == "tiny"
    cfg = load_config(os.path.join(spec["config_dir"], "env.yaml"))
    assert cfg.features_extractor == "tiny"
▶ QT_QPA_PLATFORM=cocoa .venv/bin/python -m pytest … ⟶ FAIL：AttributeError: CreateModelWizard has no attribute 'net_combo'
② 最小實作
改 build_spec 時：把原本 extra_env={"bev_fov_diag_deg": fov} 換成含 features_extractor 的版本；meta dict 內 "observation": ... 那行旁加 features_extractor。
# (1) __init__ 內、lens_combo 那幾行之後新增：
        self.net_combo = QComboBox()
        self._NET = [("原本 NatureCNN（預設）", "nature"), ("迷你 TinyCNN（~200× 小）", "tiny")]
        self.net_combo.addItems([label for label, _ in self._NET])

# (2) 在「演算法 / 超參數」頁的 rows list、("鏡頭 (BEV)", self.lens_combo) 之後加：
                                 ("策略網路", self.net_combo),

# (3) 在 observation() accessor 旁新增：
    def features_extractor(self):
        return self._NET[self.net_combo.currentIndex()][1]

# (4) build_spec：擴充 extra_env 與 meta：
                         extra_env={"bev_fov_diag_deg": fov,
                                    "features_extractor": self.features_extractor()},
        #   ...在 meta dict 內加一行：
            "features_extractor": self.features_extractor(),
▶ 同上 ⟶ PASS：預設 nature、選 tiny 時 env.yaml/meta 皆為 tiny
③ 提交
git commit -am "ui: 建模精靈可選策略網路 (NatureCNN / TinyCNN)"

def test_features_extractor_preset_round_trips(qtbot, tmp_path):
    w = _wiz(qtbot, tmp_path)
    w.apply_preset({"features_extractor": "tiny"})
    assert w.features_extractor() == "tiny"
    w.apply_preset({"features_extractor": "nature"})
    assert w.features_extractor() == "nature"
▶ QT_QPA_PLATFORM=cocoa .venv/bin/python -m pytest … ⟶ FAIL：AssertionError（apply_preset 還沒預選 net_combo）
② 最小實作
# (1) preset_from_model(model)：在回傳的 preset dict 內加：
        "features_extractor": meta.get("features_extractor", "nature"),

# (2) apply_preset(self, preset)：在 observation 預選那段旁加：
        fe = preset.get("features_extractor")
        if fe:
            i = next((k for k, (_, v) in enumerate(self._NET) if v == fe), -1)
            if i >= 0:
                self.net_combo.setCurrentIndex(i)
▶ 同上 ⟶ PASS：preset round-trip 通過
