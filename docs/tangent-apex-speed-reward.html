切點轉向 ＋ 區間速度獎勵 · Tangent Apex Speed

topRacer 獎勵函數設計規格書 · 2026-06-28 · 在 Tangent Apex 上加速度項（A 案：相加）· 後三點全速出彎

合併一檔六張表絕對 m/s 目標速區域化連續速度煞車→彎→出彎(全速)→直線

一句話：在切點轉向獎勵（0/1）之上，加區間化速度項 v∈[0,1]： 入彎前煞車(2.0) → 彎中控速(2.5) → 後三點全速出彎(4.0) → 直線衝頂速(4.0)。最終 reward = W_STEER·steer + W_SPEED·v（預設 1+1，範圍 0–2）。出彎＝全速，與直線同公式（簡化）；減速只在煞車區生效。

1. 組成

單一檔 reward_functions/Tangent Apex Speed.py，重用 Tangent Apex 全部函式（build_cache · next_corner · apex_polyline · tangent_point · bearing · normalize_deg · reward_for…），再加 braking_zone · exit_zone · speed_term，兩項相加。

3. 區間 → 目標速（絕對 m/s；優先序 煞車 > 出彎 > 彎中 > 直線）

區間條件目標 / 公式
煞車區nxt ∈ brakeBRAKE_SPEED = 2.0（峰）
出彎區nxt ∈ exit全速 4.0（同直線 speed/4）
彎中nxt ∈ left ∪ rightCORNER_SPEED = 2.5（峰）
直線其他STRAIGHT_SPEED = 4.0（speed/4）
速度常數要與模型動作空間 speed_high 一致（此處頂速 4 m/s 為例）。出彎不再有獨立常數＝直接用 STRAIGHT_SPEED。

4. 速度項 v（區域化、連續、不打架）

  • 直線 / 出彎：v = clip(speed / STRAIGHT_SPEED, 0, 1) —— 越快越高（封頂 1）。出彎沿用此式＝鼓勵全速衝出。
  • 煞車 / 彎中：v = clip(1 − |speed − target| / TOP, 0, 1) —— 在目標速達峰，過快過慢都降。
  • 關鍵：減速只在 brake 區；exit 區用直線公式，所以彎尾鼓勵加速到全速。

5. 合併（A 案：相加）

reward = W_STEER·steer + W_SPEED·v，W_STEER = W_SPEED = 1.0；steer∈(0, 1)、v∈[0,1]，最高 2.0。

取捨：相加 → 直線上轉向暫時不對仍能拿速度分（刻意，訊號穩、好訓練）；要更嚴可改乘法閘門 v·(0.3+0.7·steer)，本規格走 A 案。

6. reward_function 流程（虛擬碼）

STRAIGHT_SPEED, CORNER_SPEED, BRAKE_SPEED = 4.0, 2.5, 2.0   # 對齊 speed_high；出彎=全速同直線
TOP = STRAIGHT_SPEED          # 速度項的正規化分母
W_STEER = W_SPEED = 1.0       # A 案：相加


def braking_zone(entries, n):           # 煞車區：每個入口 e-3..e+2 共 6 點
    z = set()
    for e in entries:
        for k in range(-3, 3):
            z.add((e + k) % n)
    return z


def exit_zone(corners, n):              # 出彎區：每個彎最後 3 點 end-2..end
    z = set()
    for s, e, d in corners:
        for k in (0, 1, 2):
            z.add((e - k) % n)
    return z


def _clip01(x):
    return 0.0 if x < 0.0 else (1.0 if x > 1.0 else x)


def speed_term(speed, i, c):
    """區域化連續速度項 v∈[0,1]。優先序 煞車 > 出彎 > 彎中 > 直線。"""
    if i in c["brake"]:
        return _clip01(1.0 - abs(speed - BRAKE_SPEED) / TOP)    # @2.0 入彎前煞車
    if i in c["exit"]:
        return _clip01(speed / STRAIGHT_SPEED)                  # 後三點全速出彎（同直線）
    if i in c["left"] or i in c["right"]:
        return _clip01(1.0 - abs(speed - CORNER_SPEED) / TOP)   # @2.5 彎中控速
    return _clip01(speed / STRAIGHT_SPEED)                      # 直線：越快越高


def reward_function(params):
    global _CACHE
    if _CACHE is None:
        _CACHE = build_cache(params["waypoints"], params["track_width"])  # 含 brake / exit
    c = _CACHE
    steering = float(params["steering_angle"]); speed = float(params["speed"])
    _, nxt = params["closest_waypoints"]

    corner = next_corner(int(nxt), c["corners"], c["n"])        # 轉向項（切點，沿用 Tangent Apex）
    if corner is None:
        steer = reward_for(0.0, steering)
    else:
        P = (float(params["x"]), float(params["y"]))
        T = tangent_point(P, apex_polyline(corner, c["apex"], c["n"]), corner[2] == "L")
        e = normalize_deg(bearing(P, T) - float(params["heading"]))
        steer = reward_for(e, steering)

    v = speed_term(speed, int(nxt), c)
    return W_STEER * steer + W_SPEED * v       # A 案：相加，範圍 0..2

7. 邊界情況

情況處置
無彎（SAMPLE_PARAMS 兩點）corner=None → 直行；不在任何區 → 直線速度項；回傳數字、過 verifier。
區間重疊（短彎：入口屬煞車、彎尾屬出彎）優先序 brake > exit > corner > straight，於 speed_term 以 if 次序保證。
速度 0 / 超頂速_clip01 夾 [0,1]。
離道 / 倒車不額外處理（同 Tangent Apex）。
